# Product Requirements Document

## What the App Does
- Observe which records of cctv cameras are in archive
- Navigate thorough the records
- Play selected records

## Who It's For
- Small business owners managing inventory
- Admins overseeing cctv records and reports

## Tech Stack & Constraints

## Features & Acceptance Criteria

### Feature 1: [Initial Dashboard]
- [ ] User can open dashboard and view cameras, which at least once has a record
- [ ] System responds with cameras list, number of records sorted by time

### Feature 2: [Weekly list]
- [ ] User can select one or mere cameras and dive in the records
- [ ] For selected cameras system will return week view calendar with records form cameras
